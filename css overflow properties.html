📦 CSS overflow Properties

🔹 overflow क्या होता है?

जब किसी box (जैसे div, container) के अंदर का content उसकी fixed width या height से बड़ा हो जाता है, तब extra हिस्सा बाहर निकलने लगता है। उसी extra हिस्से को control करने के लिए overflow property का उपयोग किया जाता है।

1️⃣ overflow: visible;

overflow: visible;

📌 यह default value है।

👉 content बाहर भी दिखाई देगा
👉 कोई scroll bar नहीं आएगा
👉 content कट नहीं होगा

🧠 याद रखने की ट्रिक:

visible = सब दिखाओ, चाहे बाहर निकले
