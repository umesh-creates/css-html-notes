🔥 CSS position Property

position property से हम तय करते हैं कि कोई element page पर कैसे और कहाँ place होगा।

1️⃣ static (Default)

👉 हर element का default position: static; होता है।
👉 top, bottom, left, right, z-index काम नहीं करते।
👉 Normal document flow follow करता है।

.box{
  position: static;
  top: 20px;   /* कोई effect नहीं */
}
👇👇👇
static element को manually move नहीं कर सकते।

2️⃣ relative

👉 Element अपनी original जगह से move होता है।
👉 लेकिन उसकी पुरानी जगह खाली नहीं होती (space reserve रहता है)।
👉 top, left, right, bottom काम करते हैं। 

.box{
  position: relative;
  top: 20px;
  left: 30px;
}

🔎 समझो:
👉 Element visually move होता है
👉 लेकिन layout disturb नहीं होता

📌 Important Use:
👉👉👉 Parent को position: relative देते हैं ताकि उसके अंदर का absolute element सही जगह position हो सके।

3️⃣ absolute

👉 Element normal flow से बाहर हो जाता है।
👉 यह अपने nearest positioned ancestor के अनुसार position होता है।
👉 अगर कोई positioned parent नहीं है → तो body के अनुसार position होगा। 

.parent{
  position: relative;
}

.child{
  position: absolute;
  top: 0;
  right: 0;
}

📌 Important:
👉👉👉 Nearest positioned ancestor मतलब जिसका position static ना हो (relative, absolute, fixed, sticky).

💀 Common Bug:
👉👉👉 अगर parent को position: relative नहीं दिया तो absolute element गलत जगह चला जाएगा।

4️⃣ fixed

👉 Element viewport के अनुसार fix हो जाता है।
👉 Scroll करने पर भी अपनी जगह से नहीं हिलता। 

.navbar{
  position: fixed;
  top: 0;
  width: 100%;
}

📌 Use:
▶ Sticky navbar
▶ Floating button
▶ Chat icon

😂 Common Problem: fixed element नीचे के content के ऊपर आ सकता है।

Solution:

body{
  padding-top: 60px;
}

5️⃣ sticky

👉 यह relative और fixed का combination है।
👉 scroll होने तक relative जैसा behave करता है।
👉 एक point के बाद fixed जैसा behave करता है। 

.header{
  position: sticky;
  top: 0;
}

📌 Condition:
👉 Parent के अंदर scroll होना चाहिए।
👉 top देना जरूरी है।

😂 Common Bug:
👉अगर parent पर overflow: hidden है तो sticky काम नहीं करेगा।
👉 Parent की height कम होगी तो भी problem होगी।

🔥 Very Important Concept – z-index

👉 z-index stacking order control करता है।
👉 केवल positioned element पर काम करता है (static पर नहीं)। 

.box{
  position: relative;
  z-index: 10;
}

Bada z-index→ ऊपर दिखाई देगा।

🤣 Hidden Bugs (बहुत Important)

1️⃣ transform, filter, perspective Issue

अगर parent पर ये properties लगी हैं:
  • transform
  • filter
  • perspective
 तो वह parent एक नया containing block बना देता है।

इससे क्या होगा?
  • अंदर का position: fixed element viewport की जगह parent के अनुसार behave करेगा।
Example: 
.parent{
  transform: scale(1);
}

.child{
  position: fixed;
  top: 0;
}

👉 यहाँ fixed element viewport से नहीं बल्कि parent से relate करेगा।
👉 यह बहुत बड़ा hidden bug है।

2️⃣ overflow Problem

अगर parent पर: 
overflow: hidden;

तो:
  • absolute element cut हो सकता है।
  • sticky काम नहीं करेगा।

3️⃣ Height Collapse Problem

अगर parent में height नहीं है और अंदर absolute element है:
तो parent collapse हो जाएगा।

Solution:
  • Parent को height दो
  • या clearfix technique use करो
