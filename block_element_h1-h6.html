🔷 <h1> – <h6> Elements

🔹 Proper Name:

Heading Elements

🔹 इनका work क्या है?

इनका work होता है heading (शीर्षक) लिखना।

ये 6 level के होते हैं:

<h1> → सबसे बड़ा heading
<h2>
<h3>
<h4>
<h5>
<h6> → सबसे छोटा heading

⚠️ ध्यान:

<h1> सबसे important होता है (SEO में भी)।

🎨 Heading पर लगने वाली Important CSS Properties

1️⃣ color

👉 work:

Text का color बदलता है। 

color: 🟥red;

2️⃣ background-color

👉 work:

Heading के पीछे background देता है। 

background-color: 🟨yellow;

⚠️ यह पूरी width cover करता है क्योंकि heading block element है।

3️⃣ font-size

👉 work:

Text की size control करता है।

font-size: 40px;

Default sizes (approx):

h1 → 32px
    h2 → 24px
    h3 → 18px

4️⃣ font-weight

👉 work:

Text को bold / normal करता है।

font-weight: normal;

⚠️ Default में headings bold होती हैं।

5️⃣ text-align

👉 work:

Text alignment change करता है।

text-align: center;
        text-align: right;

6️⃣ text-transform

👉 work:

Text को uppercase / lowercase बनाता है।

text-transform: uppercase;

7️⃣ letter-spacing

👉 work:

Letters के बीच की spacing बदलता है।

letter-spacing: 3px;

8️⃣ text-shadow

👉 work:

Text के पीछे shadow देता है।

text-shadow: 2px 2px 5px gray;

Format:

horizontal vertical blur color

9️⃣ padding

👉 work:

Heading के अंदर space देता है।

padding: 20px;

🔟 margin

👉 work:

Heading के बाहर space देता है।

margin: 20px 0;

⚠️ Browser default margin देता है headings को।

1️⃣1️⃣ border

👉 work:

Heading के चारों तरफ line देता है।

border-bottom: 3px solid red;

Modern design में अक्सर bottom border use होता है।
