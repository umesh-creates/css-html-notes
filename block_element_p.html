🔷 <p> Element

🔹 Proper Name:

Paragraph Element

🔹 <p> का work क्या है?

  • <p> का work होता है paragraph लिखना।
  • यह text को अलग block में show करता है।
  • इसके पहले और बाद में browser default margin देता है।

Example:

<p>यह एक paragraph है।</p;
<p>यह दूसरा paragraph है।</p>

🎨 <p> पर लगने वाली Important CSS properties

अब हम design properties समझेंगे।

1️⃣ color

👉 इसका work:

Text का color change करता है।

color: 🟥red;

2️⃣ background-color

👉 इसका work:

Paragraph के पीछे background color देता है।

⚠️ ध्यान:

यह content + padding area को color करता है।

background-color: 🟨yellow;

3️⃣ font-size

👉 इसका work:

Text की size change करता है।

font-size: 20px;

4️⃣ font-weight

👉 इसका work:

Text को bold या normal करता है।

font-weight: bold;

या

font-weight: 700;

5️⃣ font-family

👉 इसका work:

Text का font style change करता है।

font-family: Arial, sans-serif;

6️⃣ text-align

👉 इसका work:

Text alignment change करता है।

text-align: center;
text-align: right;
text-align: left;
text-align: justify;

7️⃣ line-height

👉 इसका work:

Lines के बीच की vertical spacing control करता है।

line-height: 1.6;

यह paragraph को readable बनाता है।

8️⃣ letter-spacing

👉 इसका work:

Letters के बीच की spacing change करता है।

letter-spacing: 2px;

9️⃣ text-transform

👉 इसका work:

Text को uppercase / lowercase में बदलता है।

text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;

🔟 padding

👉 इसका work:

Paragraph के side में space देता है।

padding: 20px;

1️⃣1️⃣ margin

👉 इसका work:

Paragraph के बाहर space देता है।

margin: 20px;

⚠️ ध्यान:

Browser <p> को default margin देता है।

1️⃣2️⃣ border

👉 इसका work:

Paragraph के चारों तरफ line/border देता है।

border: 2px solid black;
