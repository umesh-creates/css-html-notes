HTML/CSS SELECTORS

🔥 1️⃣ Universal Selector

*{
    margine:0;
}

👉 सभी elements (body, p, h1, div… सब) का margin 0 हो जाएगा।

🔥 2️⃣ Element (Type) Selector

p {
  color: blue;
}

👉 सभी <p> का text blue हो जाएगा।

🔥 3️⃣ Class Selector

.box {
  background-color: yellow;
}

👉 जिन elements में class="box" है, उनका background yellow होगा।

🔥 4️⃣ ID Selector 

#main {
  color: red;
}

👉 जिस element का id="main" है, उसका text red होगा।

🔥 5️⃣ Grouping Selector

h1, h2, p {
  font-family: Arial;
}

👉 सभी <h1>, <h2> और <p>में Arial font लग जाएगा।

🔥 6️⃣ Descendant Selector (space)

div p {
  color: green;
}

👉 div के अंदर जितने भी <p> हैं, उनका text green होगा।

🔥 7️⃣ Child Selector (>)

div > p {
  color: orange;
}

👉 सिर्फ div के direct child <p> orange होंगे।

🔥 8️⃣ Adjacent Sibling (+)

h1 + p {
  color: purple;
}

👉 h1 के तुरंत बाद वाला पहला <p> purple होगा।

🔥 9️⃣ General Sibling (~)

h1 ~ p {
  color: brown;
}

👉 h1 के बाद आने वाले सभी <p> brown हो जाएंगे।

🔥 🔟 Attribute Selector (Exist)

input[required] {
  border: 2px solid red;
}

👉 जिन input में required attribute है, उनका border red होगा।

🔥 1️⃣1️⃣ Attribute Exact Match

input[type="text"] {
  background-color: lightgray;
}

👉 सिर्फ type="text" वाले input lightgray होंगे।

🔥 1️⃣2️⃣ Attribute Starts With (^=)

a[href^="https"] {
  color: green;
}

👉 जिन links का href https से शुरू होता है, वे green होंगे।

🔥 1️⃣3️⃣ Attribute Ends With ($=)

a[href$=".pdf"] {
  color: red;
}

👉 जिन links का अंत .pdf से होता है, वे red होंगे।

🔥 1️⃣4️⃣ Attribute Contains (*=)

a[href*="google"] {
  color: blue;
}

👉 जिन links में google word है, वे blue होंगे।

🔥 1️⃣5️⃣ Pseudo-Class :hover

button:hover {
  background-color: black;
}

👉 जब mouse button पर जाएगा, background black हो जाएगा।

🔥 1️⃣6️⃣ Pseudo-Class :focus

input:focus {
  border: 3px solid blue;
}

👉 जब input पर click करेंगे, border blue हो जाएगा।

🔥 1️⃣7️⃣ Pseudo-Class :first-child

p:first-child {
  color: red;
}

👉 जो p अपने parent का पहला child है, वही red होगा।

🔥 1️⃣8️⃣ Pseudo-Class :last-child

p:last-child {
  color: green;
}

👉 जो p आखिरी child है, वही green होगा।

🔥 1️⃣9️⃣ Pseudo-Class :nth-child()

li:nth-child(2) {
  color: blue;
}

👉 हर list का दूसरा <li> blue होगा।

🔥 2️⃣0️⃣ Pseudo-Class :not()

p:not(.active) {
  color: gray;
}

👉 जिन <p> में class active नहीं है, वे gray होंगे।

🔥 2️⃣1️⃣ Pseudo-Element ::before

p::before {
  content: "👉 ";
}

👉 हर paragraph से पहले 👉 symbol जुड़ जाएगा।

🔥 2️⃣2️⃣ Pseudo-Element ::after

p::after {
  content: " ✔";
}

👉 हर paragraph के बाद ✔ जुड़ जाएगा।

🔥 2️⃣3️⃣ Pseudo-Element ::first-letter

p::first-letter {
  font-size: 30px;
}

👉 हर paragraph का पहला letter बड़ा हो जाएगा।

🔥 2️⃣4️⃣ :is()

:is(h1, h2) {
  color: red;
}

👉 h1 और h2 दोनों red होंगे।

🔥 2️⃣5️⃣ :has() (Advanced)

div:has(p) {
  background-color: lightblue;
}

👉 जिन div के अंदर p है, वे lightblue हो जाएंगे।
