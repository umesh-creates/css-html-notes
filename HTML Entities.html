<code> Tag

✅ <code> Tag क्या है?

<code> tag का उपयोग programming code दिखाने के लिए किया जाता है।

⭐ Basic Example

Use <code> print() </code> function in Python.

print("hello")

अगर multiple lines का code दिखाना हो तो
<pre> और <code> tag को साथ में उपयोग करते हैं। 

<pre>
<code>
function hello() {
  alert("Hello World");
}
</code>
</pre>

🔥 <code> और <pre> में अंतर

Tag उपयोग
<code> छोटा code snippet दिखाने के लिए
<pre> Spaces और line breaks को preserve करने के लिए
<pre><code> Proper multi-line code block दिखाने के लिए
