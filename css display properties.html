🎯 CSS display Property

🔹 display क्या करता है?

display तय करता है कि element:

👉 नई line लेगा या नहीं
👉 कितनी width लेगा
👉 अंदर के children कैसे arrange होंगे

1️⃣ block

📦 Examples

div, section, p, h1 default block होते हैं।

display: block;

✔️ हमेशा नई line से शुरू होगा
✔️ पूरी width ले लेता है
✔️ width और height काम करती है

🧠 याद रखो:
block = पूरा डिब्बा

2️⃣ inline

✔️ नई line नहीं लेता
✔️ जितनी जरूरत उतनी जगह
✔️ width / height काम नहीं करती

📦 Examples

span, a, strong

display: inline;

🧠 याद रखो:
inline = text जैसा behavior

3️⃣ inline-block 🔥

✔️ नई line नहीं लेता
✔️ width / height काम करती है
✔️ block + inline का mix

display: inline-block;

🧠 याद रखो:

inline-block = smart block

4️⃣ none

✔️ element पूरी तरह गायब
✔️ जगह भी नहीं लेगा
✔️ layout से हट जाएगा

display: none;

🧠 याद रखो:

none = पूरा delete जैसा effect

5️⃣ flex 🔥🔥🔥 (Most Important)

✔️ children एक line में आ जाते हैं
✔️ alignment control कर सकते हो
✔️ modern layout system

display: flex;

🧠 याद रखो:

flex = flexible layout

7️⃣ inline-flex

✔️ flex जैसा layout
✔️ लेकिन नई line नहीं लेता
✔️ जितनी जरूरत उतनी width

display: inline-flex;
🧠 inline-flex = flex लेकिन line में

8️⃣ grid

✔️ rows और columns बना सकते हो
✔️ powerful layout control
✔️ 2D layout system

display: grid;
🧠 grid = table जैसा powerful layout

8️⃣ inline-grid;

✔️ grid layout
✔️ नई line नहीं लेता
✔️ जितनी जरूरत उतनी width

display: inline-grid;
🧠 inline-grid = grid लेकिन line में

📊 Final Quick Table

display value क्या करेगा
block नई line + पूरी width
inline line में + width height नहीं
inline-block line में + width height काम
none पूरा गायब
flex flexible layout
inline-flex flex लेकिन line में
grid row + column layout
inline-grid grid लेकिन line में
