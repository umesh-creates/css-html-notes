🔥 CSS Variables (Custom Properties)

CSS में variable को custom property भी कहते हैं।
इनका उपयोग एक ही value को बार-बार लिखने से बचने के लिए किया जाता है।

1️⃣ Variable क्या होता है?

Variable एक नाम होता है जिसमें हम कोई value store करते हैं
(जैसे color, font-size, margin आदि)।

बाद में हम उसी value को अलग-अलग जगह use कर सकते हैं।

2️⃣ Variable का Syntax

🔹 Declare करने का तरीका

--variable-name: value;

🔹 Use करने का तरीका

var(--variable-name)

3️⃣ Global Variable (Universal Variable)

यह पूरी website में हर जगह use किया जा सकता है।

इसे हम :root के अंदर define करते हैं।

:root {
  --main-color: blue;
  --font-size: 20px;
}

h1 {
  color: var(--main-color);
  font-size: var(--font-size);
}

p {
  color: var(--main-color);
}

📌 याद रखने वाली बात:

  • :root = पूरी website
  • यहाँ define किया गया variable हर element में काम करेगा

4️⃣ Local Variable

यह सिर्फ एक खास element के अंदर ही काम करता है।

.box {
  --box-color: red;
  background-color: var(--box-color);
}

p {
  color: var(--box-color);  /* यह काम नहीं करेगा */
}

📌 याद रखने वाली बात:

  • .box के अंदर define किया गया variable सिर्फ .box के अंदर चलेगा
  • बाहर use नहीं कर सकते

5️⃣ Main Difference

Type कहाँ define होता है कहाँ use होता है
Global Variable :root के अंदर पूरी website में
Local Variable किसी element के अंदर सिर्फ उसी element में

6️⃣ Fallback Value (जरूरी जानकारी)

अगर variable नहीं मिले तो backup value दे सकते हैं:

color: var(--text-color, black);

अगर --text-color नहीं मिला तो black apply होगा।
