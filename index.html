✅ 1️⃣ Block Elements (Block Level Elements)

📌 Block elements हमेशा new line से start होते हैं और पूरी width ले लेते हैं।

🔰 Common Block Elements List
Elements Use
<div> Container
<p> Paragraph
<h1> - <h6> headings
<form> Form
<table> Table
<ul> Unordered list (bullet list)
<ol> Ordered list (number list)
<li> List Item
<section> Page Section
<article> Article or Independent Content
<header> Page or Section's Header
<footer> Page or Section's Footer
<nav> Navigation Links
<aside> Aside Content
<fieldset> Wrap Group of Form
<hr> Horizontal Line
<pre> Preformatted Text Box

✅ 2️⃣ Inline Elements

📌 Inline elements new line start नहीं करते। ये सिर्फ उतनी जगह लेते हैं जितनी ज़रूरत हो।

🔰 Common Inline Elements List
Elements Use
<span> Inline container
<a> Insert Link
<img> Insert Image
<strong> Important Text (bold)
<b> Bold Text
<em> Emphasized text (italic)
<i> Italic text
<u> Underline text
<label> Form input label
<input> Input field
<select> Dropdown
<textarea> Multi-line input
<button> Button
<small> Small Text
<sub> Subscript
<sup> Superscript
<br> Break line
