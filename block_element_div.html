🔷 <div> Element

🔹 Proper Name:

Division Element

🔹 <div> का work क्या है?

  • div एक container element है।
  • इसका work होता है दूसरे elements को group करना और उन पर CSS design apply करना।

  • यह अपने आप कोई design नहीं देता।
  • यह सिर्फ एक box create करता है।

🎨 <div> पर लगने वाली important CSS properties

अब हम सिर्फ design वाली properties समझेंगे।

1️⃣ background-color

👉 इसका work:

यह element के अंदर का background color बदलता है।

⚠️ ध्यान:

  • यह सिर्फ content + padding area को color करता है।
  • margin area को color नहीं करता।
background-color: 🟥red;

2️⃣ width

👉 इसका work:

यह box की horizontal size तय करता है।

width: 300px;

या

width: 50%;

3️⃣ height

👉 इसका work:

यह box की vertical size तय करता है।

height: 200px;

⚠️ ध्यान:

अगर content ज्यादा हुआ तो overflow हो सकता है।

4️⃣ padding

👉 इसका work:

यह content और border के बीच की अंदर वाली space देता है।

padding: 20px;

यह background area के अंदर आता है।

5️⃣ border

👉 इसका work:

यह element के चारों तरफ line बनाता है।

border: 5px solid black;

Format:

border: width style color;

6️⃣ margin

👉 इसका work:

यह element के बाहर की space देता है।

margin: 20px;

⚠️ ध्यान:

margin का कोई color नहीं होता।

7️⃣ border-radius

👉 इसका work:

यह corners को गोल बनाता है।

border-radius: 10px;

पूरी तरह गोल:

border-radius: 50%;

8️⃣ box-shadow

👉 इसका work:

यह element को shadow effect देता है।

box-shadow: 0 5px 15px rgba(0,0,0,0.3);

9️⃣ color

👉 इसका work:

यह text का color बदलता है।

color: white;

📦 CSS Box Model (बहुत important)

Structure हमेशा याद रखें:

      Margin
      ➡Border
      ➡Padding
      ➡Content
