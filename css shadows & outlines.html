🎯 CSS Shadows & Outlines – Complete Notes

1️⃣ box-shadow

📌 क्या करता है?

Element के बाहर shadow बनाता है।

🧠 Syntax

box-shadow: x-offset y-offset blur spread color;

📌 Basic Example

box-shadow: 5px 5px 10px black;

इसका मतलब:

  • 5px → right
  • 5px → down
  • 10px → blur
  • black → color

🎯 Parameters समझो

value मतलब
x-offset left/right shadow
y-offset up/down shadow
blur कितना soft होगा
spread shadow कितना फैलेगा
color shadow का रंग

🔥 Full Example

box-shadow: 0px 4px 15px 2px rgba(0,0,0,0.3);

💎 Inner Shadow

box-shadow: inset 0 0 10px black;

👉 inset = अंदर shadow

✨ Multiple Shadow

box-shadow:
    0 0 5px red,
    0 0 10px blue;

2️⃣ text-shadow

📌 क्या करता है?

Text के पीछे shadow देता है।

🧠 Syntax

text-shadow: x-offset y-offset blur color;

Example

text-shadow: 2px 2px 5px gray;

Glow Effect

text-shadow: 0 0 10px red;

3️⃣ outline

📌 क्या है?

Border जैसा दिखता है

लेकिन layout space नहीं लेता। 

outline: width style color;

Example

outline: 3px solid red;

👉Border vs Outline Difference

Border Outline
Space लेता है Space नहीं लेता
Box size बदलता है Box size नहीं बदलता
Radius follow करता है Radius follow नहीं करता (mostly)

🎯 Outline Offset

outline: 3px solid blue;
outline-offset: 5px;

👉 Outline box से बाहर हट जाएगा।

💎 Focus Design (Important 🔥)

Forms में professional focus effect:

outline: none;
box-shadow: 0 0 5px blue;
