🔥 CSS Display Grid Notes

1. Grid क्या है

Grid Layout एक 2-Dimensional layout system है।

मतलब:

  • rows भी control कर सकते हैं
  • columns भी control कर सकते हैं

इसलिए यह Flexbox से ज्यादा powerful layout system है।

Grid activate करने के लिए

display: grid;

या

display: inline-grid;

2. Grid Line और Grid Cell

जब grid बनती है तो उसमें lines और cells बनते हैं।

Grid Line

row और column को divide करने वाली lines।

1 | 2 | 3 | 4

ये vertical और horizontal दोनों हो सकती हैं।

Grid Cell

दो row lines और दो column lines के बीच का single box।

| cell |

एक grid item normally एक cell में रहता है।

3. Grid Track और Grid Area

Grid Track

दो lines के बीच का space।

दो प्रकार के होते हैं

type meaning
column track column की width
row track row की height

column track

example

grid-template-columns: 50px 50px 50px;

यह 3 column tracks बनाएगा 50-50px width वाले।

View

column2 = 50px
column3 = 50px

row track

example

grid-row-columns: 50px 50px 50px;

यह 3 row tracks बनाएगा 50-50px height वाले।

View

मतलब

row1 = 50px
row2 = 50px
row3 = 50px

Grid Area

जब एक item multiple cells को combine कर लेता है।

| cell | cell |
| cell | cell |

यह पूरा area grid area कहलाता है।

12. justify-content और align-content

ये पूरी grid की alignment control करते हैं।

जब container में extra space हो।

justify-content

horizontal direction

justify-content: center;

align-content

vertical direction

align-content: center;
property काम
justify-content grid को horizontal move
align-content grid को vertical move

Grid Format

Important Note (Grid Alignment)

Items को align करने के लिए grid cell के अंदर extra space होना जरूरी है। क्योंकि items की alignment grid cell के अंदर होती है, container के अंदर नहीं।

इसलिए:

  • अगर item की size = grid cell की size हो जाती है → तो align-items / justify-items का effect दिखाई नहीं देगा क्योंकि item के पास move करने की जगह नहीं बचती।
  • इसलिए अक्सर item की size grid cell से छोटी रखी जाती है, ताकि cell के अंदर extra space बने और item align हो सके।

दूसरी तरफ

  • align-content / justify-content items को नहीं बल्कि पूरे grid को move करते हैं।
  • alignment container के अंदर होती है।
  • यानी पूरा grid container के अंदर move होता है।

Short Rule

items alignment  → grid cell के अंदर होती है
content alignment → container के अंदर होती है

Flexbox vs Grid (Important)

Feature Flexbox Grid
Dimension 1D 2D
Control row या column row + column
Best for small layout page layout
